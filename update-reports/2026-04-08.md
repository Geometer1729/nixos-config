## Notable Upstream Details

### Hyprland 0.54.3
Pure bugfix release — crash fixes in compositor, layout handling, and XWayland. No breaking changes. The breaking changes (togglesplit/swapsplit removal, windowrule overhaul) were in 0.53/0.54.0, already handled in prior updates.

### Firefox 149.0
Major security release with **37 vulnerability fixes** spanning memory corruption, sandbox escapes, use-after-free flaws, and RCE risks. No breaking changes for Linux users. Windows 8.1 and macOS 10.14 support dropped (not relevant to NixOS).

### network-setup.service removal
In the scripted networking backend (default when `networking.useNetworkd = false`), `network-setup.service` is replaced by `network.target` integrated directly into `multi-user.target`. Interface addresses, routes, and gateways are now configured asynchronously as devices become available. This config uses NetworkManager so impact should be minimal, but the systemd unit changes are visible in the closure diff.
